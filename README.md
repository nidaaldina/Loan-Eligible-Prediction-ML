# Loan-Eligible-Prediction-ML

## Informasi Dataset
Data yang digunakan berasal dari Kaggle [Loan Eligible Dataset](https://www.kaggle.com/datasets/vikasukani/loan-eligible-dataset/data)

Data memiliki kolom:
1. Loan_ID
2. Gender
3. Married
4. Dependents
5. Education
6. Self_Employed
7. ApplicantIncome
8. CoapplicantIncome
9. LoanAmount
10. Loan_Amount_Term
11. Credit_History
12. Property_Area
13. Loan_Status

## Machine Learning Model
Model machine learning yang digunakan adalah:
1. Logistic Regression
2. Decision Tree
3. Random Forest
4. SVM
